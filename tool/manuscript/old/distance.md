## Distance question

Leetcode 上有很多問題是關於距離，這裡紀錄下一些有用的數學概念。

- **平均值 (mean)**

    平均值的歐幾里得距離最短 (4\*4 + 4\*4 < 1\*1 + 7\*7)

    但相乘值最大 (4\*4 > 1\*7)

- **中位數 (median)**

    中位數的絕對偏差值最低 (跟其物件的距離之總和最短)

- **眾數 (mode)**

    mode minimizes distance for indicator function (理解不能)

    這個我沒寫過類似的，寫過後我再更新。
